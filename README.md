# Robotics Production Data

A comprehensive dataset collection documenting global robotics and related technology production metrics across multiple categories.

## Overview

This repository contains curated datasets tracking production volumes, shipment statistics, and market trends for various robotics and technology categories. The data spans from the mid-1970s to present day, providing valuable historical context for understanding technological adoption and market evolution.

## Dataset Categories

### Robotics Types
- **Drones** - Unmanned aerial vehicle production and shipment data
- **Humanoids** - Humanoid robot production metrics
- **Mobile Manipulators** - Combined mobility and manipulation systems
- **Quadrupeds** - Four-legged robotic platforms
- **Robotic Arms** - Industrial and collaborative robotic arms
- **Wheeled Robots** - Mobile wheeled robotic systems

### Technology Markets
- **PC Shipments** - Personal computer annual shipments and market analysis (1975-2025)
- **Smartphone Shipments** - Global smartphone production volumes (2005-2025)

## Data Format

All datasets are provided in CSV (Comma-Separated Values) format for easy integration into analysis tools, databases, and visualization platforms.

## Usage

These datasets are suitable for:
- Market analysis and trend forecasting
- Historical technology adoption studies
- Supply chain and production planning
- Academic research in robotics and computing
- Business intelligence and competitive analysis

## Data Sources

Each dataset includes source attribution and notes fields documenting the origin and reliability of the data points.
